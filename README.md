## Installation ⚙️

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/github-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd github-portfolio
   ```
